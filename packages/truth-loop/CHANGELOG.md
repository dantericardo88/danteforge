# Changelog

All notable changes to `@danteforge/truth-loop` will be documented in this file.

## [1.0.0] - 2026-04-29

### Added

- Portable Truth Loop v1 schema types for runs, artifacts, evidence, verdicts, next actions, and budget envelopes.
- ID helpers used by DanteForge and sister repos.
- Proof helpers that attach `@danteforge/evidence-chain` bundles to artifacts, evidence records, and verdicts.
