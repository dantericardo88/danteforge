# Magic Levels

DanteForge Magic presets provide tiered intensity for agentic workflows, balancing token efficiency with depth.

## Preset Levels

- **/spark**: Light planning (low tokens, quick spec).
- **/ember**: Balanced default (medium depth, standard OSS scan).
- Follow-up PRD gap closing with targeted refinements — **/magic**
- **/blaze**: High-intensity execution waves (multi-agent party mode).
- First-time new matrix dimension + fresh OSS discovery (full autoresearch harvest) — **/inferno**

Use `--magic-level <level>` or slash commands in supported hosts (Claude Code, Codex, Cursor).

For maximum token savings, start with /ember and escalate only on stalls.
